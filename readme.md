# Local Dev

## Backend App

1. `cd backend`
2. `pip install -r requirements.txt`
3. `cp .env.sample .env`
   * There are preset variables available to test scenarios such as:
     * Has redis app
     * Has scaled environment
     * Set staging/production
4. `python main.py`

## Frontend App
1. `npm install`
2. `npm run start`
